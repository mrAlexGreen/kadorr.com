Примеры

Без клика

$('.toggle').toggles({clickable:false});

Без перетаскивания

$('.toggle').toggles({dragable:false});

Кликабельный элемент

Нажмите тут!
$('.toggle').toggles({click:$('.clickme')});

Переключение чекбокс


$('.toggle').toggles({checkbox:$('.checkme')});

Выключен по умолчанию

$('.toggle').toggles({on:false});

Пользовательский текст

$('.toggle').toggles({ontext:'Привет',offtext:'Пока'});
